# 🩺 Diabetes Disease Progression Prediction using Machine Learning

A Machine Learning project that predicts **diabetes disease progression** using the **Scikit-learn Diabetes Dataset**. The project demonstrates the complete machine learning workflow, including data preprocessing, exploratory data analysis (EDA), feature selection, model training, and performance evaluation using **Linear Regression**.

---

## 📖 Project Overview

This project aims to predict the progression of diabetes based on various medical attributes of patients. A Linear Regression model is trained on the Diabetes dataset after performing data preprocessing and feature selection to estimate disease progression accurately.

---

## 🎯 Objective

* Analyze the Diabetes dataset.
* Perform Exploratory Data Analysis (EDA).
* Select the most relevant features.
* Train a Linear Regression model.
* Evaluate model performance using regression metrics.
* Visualize prediction results.

---

## 📊 Dataset Information

The project uses the **Diabetes Dataset** available in the **Scikit-learn** library.

**Dataset Statistics**

* 📄 Total Samples: **442**
* 📌 Features: **10**
* 🎯 Target: Diabetes Disease Progression Score

### Features

* Age
* Sex
* Body Mass Index (BMI)
* Blood Pressure (BP)
* Serum Measurement S1
* Serum Measurement S2
* Serum Measurement S3
* Serum Measurement S4
* Serum Measurement S5
* Serum Measurement S6

1. Clone this repository.


2. Navigate to the project directory.


3. Install the required libraries.


4. Open the notebook.

